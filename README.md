# ClassExchange

Sistema frontend para gestão de troca e substituição de aulas no IFMS. Focado em produtividade, clareza visual e fluxo simples para professores e coordenação.

## Visão Geral
- Dashboard com visão de métricas e atividades recentes
- Módulos de Troca de Aula e Substituição com estados de carregamento
- Notificações e Solicitações com rotas dedicadas e páginas de parecer
- Design técnico com cantos marcados, rótulos “tech-label” e componentes consistentes

## Principais Rotas
- `/` Landing page
- `/login` Autenticação
- `/dashboard` Painel administrativo
- `/dashboard/professor` Painel do professor
- `/dashboard/troca` Troca de aula
- `/dashboard/substituicao` Substituição de aula
- `/dashboard/solicitacoes` Lista de solicitações
- `/dashboard/solicitacoes/[id]` Parecer de solicitação
